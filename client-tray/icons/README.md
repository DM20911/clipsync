# Tray Icons

Place 18×18 PNG (preferably 36×36 @2x) in this directory:

- `connected.png` — green dot or full color
- `disconnected.png` — gray dot
- `paused.png` — yellow / pause symbol
- `error.png` — red / warning

Plus `icon.icns` (macOS), `icon.ico` (Windows), `icon.png` 512×512 (Linux) for app installer.

Until you provide real assets, you can use any placeholder PNG; tray will still work but show a blank.
